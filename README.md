**Preserved source for Club Penguin Plus — shared for the community.**

</div>

---

## What this organization is

This is an **archived mirror** of work that was once part of **Club Penguin Plus**, a fan project around Club Penguin–style gameplay and tooling.

The project was **not open source from day one**. I contributed **before** it ever went public as open source; **later** the codebase was released under an open model. After the original public repository was **removed**, a copy of the tree was **recovered and archived** here so people can still read, learn from, and run it locally in a responsible way.

## Repositories

| Repo | Role |
|------|------|
| [**cpplus-server**](https://github.com/cpplus-archive/cpplus-server) | Node.js backend: Express, Socket.IO, Sequelize — worlds, handlers, `cpplus.sql` / analytics schema in-repo. |
| [**cpplus-client**](https://github.com/cpplus-archive/cpplus-client) | Web client: Phaser 3, Webpack, application source under `src/`. |
| [**cpplus-assets**](https://github.com/cpplus-archive/cpplus-assets) | Static game payload: fonts, media, scripts, styles — usually pulled as a **submodule** at `assets/` inside the client. |
